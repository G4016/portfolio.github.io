Project 5

Electronic Voting Machine

Designed and developed a secure Electronic Voting Machine using an AVR ATmega microcontroller, programmed entirely in Embedded C. The system supports voter authentication via a 4-digit PIN using a keypad, with real-time feedback provided through an LCD display.

We covered two test cases:
1) Majority Voting: In case if a party gets maximum votes the result is directly displayed.
2) In case of equal votes by every party , the message is displayed as "Equal Votes" and the party that has registered first is elected as the winner.

We also covered two modes of result display:
1) Master mode: Can access EVM through different ID allocated(7676) to avoid multiple voting.
2) Voter mode: Can vote using different ID allocated (eg 4016).

Technologies Used: AVR AtMega Keypad interfacing LCD display integration
