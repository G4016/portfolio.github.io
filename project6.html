Project 6

Yellove Bus Booking Assistant

Designed and implemented a bus seat booking system that allows users to register, login, book a ticket based on source and destination obviously on availability of seats and accordingly displays list of travelling passengers.

Key features:

Booking Management:
(1) Checks for already-booked seats.
(2) Displays available buses and their timings.
(3) Manages user login, logout, and ticket display.
Student Discount: Gives a discount of 10% of the trip expense if user claims to be a student.

Techniques Used: Object Oriented Programming C++ Classes and Objects Inheritance Exception Handling Virtual Functions & Polymorphism
